Welcome to Just Another Anti-Microbial Peptide database!

To date, there are more than ten Anti-Microbial Peptide databases that have been set up by different laboratories around the world (See Other Resources for a listing of some of the databases), but no serious effort has been taken to curate and consolidate the data present from these disparate databases to generate a non-redundant database. Thus, this database purports to be the first near-comprehensive non-redundant database for antimicrobial peptide sequences. We also provide bibliographical information, and cross references where available, for the interested reader.

Accordingly, using the non-redundant sequence database that we set up here, we also provide a BLAST interface for researchers to perform a BLAST query to identify homologues based on their peptide sequence of interest. We hope that this website, and the non-redundant sequence database, will be an invaluable tool that will facilitate your research!

Specific features of the database includes

  • Links to other AMP databases
  • Identify closest AMP given an unknown sequence
  • Query for an AMP of interest
