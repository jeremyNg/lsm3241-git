There are numerous other databases that are available for access by researchers. Much of the data that are present in this database are in fact consolidated and curated from these other sources. The list below, while not exhaustive, provides the sites of some of the more major databases that contain information on AMPs:

  • AMPer : A Database and an Automated Discovery Tool for Gene-Coded Antimicrobial Peptides
  • Collection of antimicrobial peptides (CAMP)
  • DAMPD : Dragon Antimicrobial Peptide Database
  • Defensins Knowledgebase
  • LAMP: A database linking antimicrobial peptide.
  • PenBase : The Shrimp Antimicrobial Peptide Penaeidin Database
  • Peptaibol Database
  • RAPD : Recombinantly-produced Antimicrobial Peptides Database
  • The Antimicrobial Peptide Database (APD)

As at the point of consolidating this database, these external databases were known to be functional. We, however, are not responsible for the accuracy of the data of these other databases. To the best of our abilities, we have validated the data from each source by cross-referencing. If you will like to obtain the raw data as they are in our database, please feel free to write to us, and we will certainly be glad to provide you access to our raw data.
